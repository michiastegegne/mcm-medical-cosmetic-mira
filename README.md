# MCM Medical Cosmetic Mira

Mehrseitige statische Website fuer MCM Medical Cosmetic Mira in Laufen.

## Lokal ansehen

```bash
python3 -m http.server 4173
```

Danach im Browser oeffnen:

```text
http://127.0.0.1:4173
```

## Noch zu ergaenzen

- echte Telefonnummer oder WhatsApp-Link
- echte E-Mail-Adresse, falls gewuenscht
- finale Studiofotos statt der aktuellen Platzhalterbilder
- finale Preise
- echte Kundenbewertungen statt Beispieltexten
- rechtliche Detailtexte fuer Impressum und Datenschutz
- optional: echtes Kalender-/Buchungsbackend statt der aktuellen Termin-Anfrage im Browser
